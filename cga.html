California's Great America

Located in Santa Clara, California's Great America features world-class roller coasters and family attractions. From the iconic Flight Deck to the thrilling Gold Striker wooden coaster, this park offers something for every thrill seeker.

Major Coasters

Flight Deck

True Intensity Scale: 8.5
‚ñº

Flight Deck is an inverted roller coaster that takes riders through intense loops and corkscrews while suspended beneath the track. The ride features a dramatic first drop and multiple inversions that provide an adrenaline-pumping experience for thrill seekers.

Community Highlight: "The view of the bay area from the top of the first drop is absolutely breathtaking, and the zero-g roll is perfectly executed!"

Gold Striker

True Intensity Scale: 7.8
‚ñº

Gold Striker is a wooden roller coaster that delivers classic airtime and thrilling drops. Built by Great Coasters International, this coaster features a traditional out-and-back layout with plenty of airtime hills and banked turns that showcase the beauty of wooden coaster engineering.

Community Highlight: "The airtime on this wooden coaster is absolutely insane! Best wooden coaster in California hands down."

Demon

True Intensity Scale: 6.2
‚ñº

Demon is a classic Arrow Dynamics looping coaster that features multiple inversions including loops and corkscrews. This steel coaster provides a smooth ride experience with intense G-forces and classic coaster elements that have stood the test of time.

Community Highlight: "A classic Arrow looper that still delivers great thrills. The double loops are perfectly executed!"

Patriot

True Intensity Scale: 7.1
‚ñº

Patriot is a floorless roller coaster that provides riders with an unobstructed view of the track ahead. The ride features smooth inversions and a layout that maximizes the thrill factor while maintaining rider comfort throughout the experience.

Community Highlight: "The floorless design makes every element feel more intense. Great for families and thrill seekers alike!"
